# Limits and Ethics

The connector is for public, policy-gated evidence retrieval.

## Limits

- public page access only
- no login or private data
- no attachment/download harvesting
- no internal search endpoint crawling
- conservative rate limits
- source URLs preserved
- derived packets remain weaker than source pages

## Ethics

The project should reduce repeated manual searching while respecting the source
site's boundaries and load. When policy is uncertain, stop and ask for review.
